PROYECTOS ACTIVOS

Laboratorio de sistemas propios

Proyectos personales en distintas etapas: algunos ya están en uso local, otros están en validación, documentación o desarrollo incremental.

Activo IA local

PolyLLM Router

Router local/API para coordinar modelos locales y externos. Usa Qwen como ejecutor principal, planificación híbrida, perfiles de costo, agentes controlados y validaciones offline.

  • Qwen local como executor final
  • DeepSeek/Kimi como planner opcional
  • Rutas híbridas, control de costo y metadata
Activo Qwen

Qwen Code Core

Entorno local para trabajar con Qwen en tareas de código, chat interactivo, contexto de proyecto, validación de respuestas y perfiles de runtime sobre llama.cpp.

  • CLI propia para prompts de código
  • Contexto automático con presupuesto
  • Perfil large validado con contexto extendido
Activo Infraestructura

NeuroFleet

Sistema personal para organizar entornos técnicos, validar configuraciones, documentar capacidades y mantener flujos seguros de automatización sin exponer detalles internos.

  • Inventario JSON validado
  • Workers GPU y perfiles por nodo
  • Dry-runs protegidos por guards
Estable Web app

GymControl

Aplicación web orientada a gestión operativa. Integra frontend, backend, base de datos, estados trazables y lógica de confirmación para flujos reales.

  • Frontend + backend + persistencia
  • Estados operativos y validación
  • Preparación para despliegue
En diseño Seguridad

Antivirus / Hardening Linux

Proyecto de herramientas para análisis local, escaneo, hardening, monitoreo básico y documentación de seguridad en equipos Linux.

  • Escaneos locales y reportes
  • Checklist de hardening
  • Automatización de auditorías simples
En desarrollo IA aplicada

Traductor IA

Traductor asistido por IA para flujos de texto, documentación, prompts y adaptación de lenguaje técnico entre español e inglés.

  • Traducción técnica
  • Normalización de tono
  • Preparado para integración con modelos locales

# Portfolio Site

Sitio estatico para compartir en LinkedIn y CV.

## Ejecutar local
Desde la raiz del repo:

```bash
python3 -m http.server 8081 -d portfolio
```

Abrir:
- http://127.0.0.1:8081

## Publicar rapido
Opciones recomendadas:
1. Netlify (drag and drop de carpeta `portfolio/`)
2. Vercel (import del repo y set root directory a `portfolio`)
3. GitHub Pages (publicar carpeta `portfolio`)
